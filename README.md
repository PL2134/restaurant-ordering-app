# Jimmy's Diner 🍕🍔🍺

A restaurant ordering app built with vanilla HTML, CSS and JavaScript as part of the Scrimba Frontend Developer course.

## Features

- Menu rendered dynamically from a JavaScript data file
- Add and remove items from your order
- Running total calculated in real time
- Payment modal with form validation
- Order confirmation message with customer's name

## Tech Stack

- HTML
- CSS (Flexbox, Google Fonts)
- Vanilla JavaScript
- Design reference: Figma

## How to Use

1. Browse the menu and click **+** to add items to your order
2. Click **remove** next to any item to remove it
3. Click **Complete order** to open the payment modal
4. Fill in your card details and click **Pay**
5. Receive your confirmation message!
